# NCrypted-Files

This is a Python-based tool for file encryption and decryption, built using Flask and the Fernet symmetric encryption algorithm. The application allows users to securely encrypt and decrypt files, ensuring data confidentiality and integrity.

## Features

- **Encryption**: Securely encrypt files with a unique key.
- **Decryption**: Easily decrypt previously encrypted files using the correct key.
- **Web Interface**: User-friendly web interface built with Flask for ease of use.

## Technologies Used

- **Python**: Programming language for backend logic.
- **Flask**: Web framework for building the web interface.
- **Cryptography**: Library that includes the Fernet module for encryption.
