Troubleshooting Guide

If you're seeing "Invalid key" or similar errors:

  • Check for leading/trailing spaces in the key
  • Ensure the key is exactly 44 characters long
  • Verify you're using the correct key for this specific file
  • Remember that keys are case-sensitive

If the file appears corrupted after decryption:

  • The file may have been encrypted with a different tool
  • The encrypted file might have been corrupted during transfer
  • Try downloading the encrypted file again
  • Ensure the file wasn't modified after encryption
Important Security Note

Without the correct encryption key, your files cannot be recovered. There is no "master key" or backdoor. Always store your keys securely!
