Troubleshooting

If you see "Invalid key" error:

  • Check for extra spaces before/after the key
  • Ensure the key is exactly 44 characters
  • Verify you're using the correct key for this file
  • Keys are case-sensitive

If the file appears corrupted after decryption:

  • The file may have been encrypted with a different tool
  • The encrypted file might be corrupted
  • Try downloading the encrypted file again
  • Ensure the file wasn't modified after encryption
Important: Without the correct encryption key, your files cannot be recovered. There is no "master key" or backdoor. Store your keys securely!
